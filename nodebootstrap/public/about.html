ANALYZR

Analyzr was born out of a need to accurately and objectively summarize the mood of articles.

As programmers, we read a lot of tech articles, and have noticed that these articles portray companies, technologies, and people with certain biases.
As readers we pick up on these sentiments, but as programmers we ask: Can computers do the same?

Analyzr allows users to input the address of an article and using natural language processing (via Alchemy API) finds the topics which are most relevant to the article. It then calculates the sentiment the article has towards each topic and presents the data to the user:

Visual Depiction

Using D3's modeling software, graphs of the topics in the article are created, showing both the topic's relevance to the article, as well as its "sentiment value", in a tooltip.

Analyzr allows filtering by subject type. This includes broad filters like "Most Relevant" and "Only Positive", and more specific ones like "Country" and "Operating System". Analyzr is able to break topics down into over 100 different categories.

Detailed Tables

For a more detailed analysis of the article, a table of the topics is displayed below the graph. Each listing contains a small breakdown of the topic, including external database links to learn more about the topic.
