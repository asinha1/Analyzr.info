ANALYZR

About Us.

Analyzr was born out of a need to accurately and objectively summarize the mood of articles.

As programmers, we read a lot of tech articles, and have noticed that many of these articles discuss the same trending topics in the tech industry. Each article has a positive/negative spin on each subject, whether that subject be a person, company, technology, or other entity.
As readers we pick up on these sentiments, but as programmers we ask: Can computers do the same? And if they can, can we utilize them to make understanding articles easier?

ANALYZR

Analyzr allows users to input the address of an article, and using natural language processing (via Alchemy API), extracts a list of topics mentioned in the article, and a detailed analysis of the article's sentiment towards those topics.
Analyzr then presents the information in a variety of ways:

Visual Depiction

Using D3's modeling software, graphs of the topics in the article are created, showing both the topic's relevance to the article, as well as its "sentiment value", in a tooltip. Analyzr allows the user to, via a drop down menu, let the user change what the graph displays, with filters, including broad ones like "Most Relevant" and "Only Positive", and more specific ones allowing you to filter by category.Analyzr is able to break topics down into over 100 different categories.
Visual relation to the relevance and sentiment via color transitions and size changes are used as an aid to the user.

Clear, Concise Tables

For a more detailed analysis of the article, a table of the topics being displayed on the screen and relevant data about them, is displayed under the graph. Each listing contains a small breakdown of the topic, useful links, and a more detailed breakdown of the sentiment and relevance scores.
